## 学習記録
### 2026-07-15：入力処理の関数化
BMI Calculatorの身長と体重の入力処理を、それぞれ独立した関数に分けました。
#### 今日行ったこと

- 身長を入力するinput_height()関数を追加
- 体重を入力するinput_weight()関数を追加
- 入力値の型変換とエラー処理を各関数内に移動
- 0より大きい値が入力された場合、returnで値を返すように変更
- main()関数の処理を整理

#### 学んだこと

- 関数を使って処理を役割ごとに分ける方法
- returnを使って関数から値を返す方法
- 関数の戻り値を変数で受け取る方法
- 入力処理を関数化すると、main()が読みやすくなること

#### 次回の改善予定

- 身長と体重で共通する入力処理を一つの関数にまとめる
- BMIの計算履歴を表示できるようにする
