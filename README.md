## 学習記録
### 2026-07-14：入力エラー処理と繰り返し計算の追加
BMI Calculatorを、誤った入力に対応し、何度でも続けて計算できるように改善しました。
#### 今日行ったこと

- try、exceptを使用して、数字以外の入力に対応
- 身長または体重が0以下の場合、計算を行わないように変更
- while文を使用し、BMIを繰り返し計算できる機能を追加
- 計算後に「もう一度計算しますか？」と確認する処理を追加
- 選択肢以外が入力された場合、再入力を求めるように変更

#### 学んだこと

- try、exceptによる例外処理
- While Trueによる繰り返し処理
- continue、break、returnの使い分け
- 条件分岐を使った入力値のチェック

#### 次回の改善予定

- 入力処理を関数として分ける
- BMIの計算履歴を表示できるようにする
