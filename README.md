### 2026-07-16：入力処理の共通化
BMI Calculatorのinput_height()とinput_weight()には同じチェック処理があったため、共通関数input_positive_number()にまとめました。
#### 今日行ったこと

- input_height()とinput_weight()を共通関数に統合
- 入力項目名と単位を引数で受け取るように変更
- 身長と体重の入力処理から重複するコードを削除
- main()から共通関数を呼び出すように変更

#### 学んだこと

- 引数を使って一つの関数の動作を切り替える方法
- 共通する処理を一つの関数にまとめる方法
- コードの重複を減らすと、修正や保守がしやすくなること
- 変数に単位を含めると、値の意味が分かりやすくなること
